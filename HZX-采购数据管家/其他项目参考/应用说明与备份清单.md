## 环境依赖

运行 / 打包本应用需要安装：

```bash
pip install PyQt5 qfluentwidgets requests cryptography pyinstaller
```

## 更新流程

1. 修改 `haozhu_fluent.py` 中的 `APP_VERSION` 版本号
2. 打包：`pyinstaller --onefile --windowed --name "HZX-智客4.0" ...`
3. 上传 EXE 到 GitHub Release
4. 用户下次启动自动检测更新
